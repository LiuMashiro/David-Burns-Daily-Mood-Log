# 伯恩斯情绪记录
## David-Burns-Daily-Mood-Log
**微信小程序：伯恩斯情绪记录**

---
一款情绪管理辅助工具，有助于缓解焦虑、抑郁、恐惧、易怒等不良情绪。

通过结构化的记录，帮助你客观评估情绪、识别消极思维中的“认知扭曲”，并通过深度追问重构更加理性和积极的思维模式。

建议在情绪波动时使用，可以缓解情绪，且长期练习可有效提升情绪管理能力。

本程序本地化优化改进了伯恩斯原版认知扭曲列表，并将原本的纸质表格优化为了适宜在移动设备上记录浏览的形式。

理论依据：大卫•伯恩斯博士《新情绪疗法》、[《伯恩斯焦虑自助疗法》（微信读书）](https://weread.qq.com/book-detail?type=1&v=6d832250721eb3ec6d8a8d8)每日情绪日志，该模型由戴维·伯恩斯(David D. Burns)1984年提出，2003年修订。戴维·伯恩斯自助疗法基于认知行为疗法（CBT），并具备临床医学意义。

## 使用流程
1.  **客观写下触发情绪的事实**  
    不带主观评价，只描述发生了什么。
2.  **选择情绪类别并评估强度**  
    识别当下的情绪（如焦虑、愤怒、难过），并为其打分（0-100）。
3.  **诊断想法中的认知扭曲并重构**  
    找出自动负性思维里的常见认知偏差，并用更理性的想法替代。
4.  **复盘对比情绪的改善情况**  
    重新评估情绪强度，观察想法调整前后的变化。

## 认知扭曲表
*经过优化，或与原书表述略有差异。*

1. **非黑即白**：
用极端的、非此即彼的二元标准评判自己或他人，无视中间地带，导致过度高要求。

2. **以偏概全**：
仅凭一次失败或一个负面事件，就得出时间或项目范围上的普遍性结论，从“单一点”错误地推广到“整个面”的普遍规律。

3. **负面滤镜**：
注意力过度关注和放大一件事情的消极点，同时可能还忽视、轻视积极的一面。

4. **主观臆断**：
在没有充分事实依据的情况下妄下结论。主要有两种表现：读心术（想当然地认为别人在负面评价自己）和预知术（在没发生前就笃定事情的结果一定是坏的）。

5. **情绪推理**：
因为自己的感受不好、情绪难受，就认为事情一定不好。

6. **过度要求**：
用"应该"、"必须"等要求性词汇要求自己或某人做某事，否则就会不满。

7. **乱贴标签**：
用简单粗暴的负面标签来定义自己或他人的全部人格。

8. **归咎指责**：
遇到问题时，不思考如何解决，而是急于找责任人，自我归罪或指责他人。
